Event Registration & Attendance Management System (Lourdes-Connect)

A modern web-based platform for managing college events, registrations, attendance, certificates, and faculty administration.

📌 Overview

The Event Registration & Attendance Management System is a full-stack web application developed to simplify the complete lifecycle of college event management. It provides a centralized platform where students can register for events, organizers can manage registrations, faculty can verify attendance, and participants can download certificates after successful completion.

The platform eliminates manual registration processes, reduces paperwork, automates attendance tracking, and provides real-time dashboards for administrators.
